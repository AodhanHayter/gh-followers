Used technologies for this project.

Angular JS 1.6.9 (Version 2 was not allowed): Used to load async data with $http service and retrive an initial list of Github Users, and a list of their followers. The list was displayed by Angular using ng-repeat just passing the response data to the $scope variables

JQuery 3.3.1 For easier DOM manipulation

Pooper 1.14.3 Required by Bootstrap slider component

Bootstrap 4 To use the new Flex capabilites in grid layout, responsive design and that beautiful slider

Sticky-Kit JS 4 For that sticky header.

Anime JS For those funny animations applied to the captions inside the slides.

- Also used some other tools or pretty effects, like smooth scrolling for links
- Photoshop CS6 to create the the logo image
- Native CSS Parallax Fixed Background
- A loading gif taken from w3schools.com to increase user experience when connection its slow
- Modals to prevent user from leaving the page and go back to the initial state list as required.

- Only found an issue with the clipboard functionality, it seems to work the first time.
