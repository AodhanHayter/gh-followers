# Followers

AngularJs project to show users from GitHub's public api.

You'll need to install [NodeJs](https://nodejs.org/es/) to run the app locally.

After NodeJs is installed type the following commands:

`npm install -g gulp-cli`

If you have any "EACCESS" error, re-run the command as administrator.

 ## Running the web application
 
 In order to run a local server showing the functionality of this app, you should run the following commands after you had pulled the code from this branch..
 
 * Run `npm install` to install the required dependencies.
 * Run `npm run serve` and wait for webpack to finish, after that you should visualize the project on your localhost:3000.
